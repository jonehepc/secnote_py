<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 段落工具栏 → QTextDocument block/list model | 用户输入改变段落结构，必须保持 Qt 文档语义和 undo 边界。 |
| 待办按钮 → HTML 输出 | 待办项必须是普通文本，不能引入可执行/交互表单控件。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-09 | Tampering | heading actions | mitigate | 使用 `QTextBlockFormat.setHeadingLevel`，不通过手写 HTML 或纯字符字号模拟标题。 |
| T-04-10 | Tampering | todo action | mitigate | 只插入普通文本 `☐ `，测试和 grep gate 禁止 checkbox/input。 |
| T-04-11 | Denial of Service | indent action | mitigate | 减少缩进使用 `max(0, current + delta)`，避免负缩进异常。 |
| T-04-12 | Repudiation | paragraph/list actions | accept | 本地单用户编辑器不记录操作审计；Qt undo 栈提供会话内可撤销性。 |
</threat_model>
