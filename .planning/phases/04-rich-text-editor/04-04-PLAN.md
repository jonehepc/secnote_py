<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| OS Clipboard → QTextEdit | 剪贴板是不可信输入，可能包含 HTML、图片、本地路径、远程 URL。 |
| QTextDocument undo stack → 页面切换 | undo 栈可能保留上一页明文片段，页面加载必须清理。 |
| View menu zoom → editor display | 缩放是显示偏好，不应进入加密数据模型。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-13 | Tampering | SafeRichTextEdit.insertFromMimeData | mitigate | 拒绝/降级包含 `<img`、`file:`、`http:`、`https:`、`<script`、event handler 属性或 `javascript:` URL 的 HTML 粘贴。 |
| T-04-14 | Information Disclosure | QTextDocument undo stack | mitigate | 本组件 `load_html` 已清理 undo/redo stacks；本计划保留并由测试覆盖。 |
| T-04-15 | Information Disclosure | zoom API | mitigate | 缩放只维护 `_zoom_steps` 内存字段，不写入 `toHtml()`、SNoteItem、Serializer 或 QSettings。 |
| T-04-16 | Denial of Service | paste HTML | accept | 本地桌面粘贴大文本由 Qt 处理；本阶段不增加额外大小限制。 |
</threat_model>
