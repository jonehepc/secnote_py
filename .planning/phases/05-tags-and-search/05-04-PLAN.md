<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 用户 query/字段选择 → SearchService | 未可信输入决定搜索范围，但只作为字符串匹配处理 |
| SearchResult.snippet → Qt rich text label | 高亮片段进入 UI 展示层 |
| SearchDialog → MainWindow | result_activated payload 将触发导航跳转 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-04-01 | T/I | result snippet display | mitigate | Dialog 只展示 `SearchService.snippet`，不读取 `note.content`，不把原始 HTML 注入 QLabel。 |
| T-05-04-02 | I | query/results logging | mitigate | 不使用 `print`/`logging` 输出 query、片段或正文；错误只显示固定文案。 |
| T-05-04-03 | D | field filter | mitigate | 至少保持一个字段勾选；空 query 不执行搜索并显示提示。 |
| T-05-04-04 | R/T | dialog lifecycle | mitigate | 结果激活只发信号，不关闭弹窗、不修改数据、不调用 dirty。 |
</threat_model>
