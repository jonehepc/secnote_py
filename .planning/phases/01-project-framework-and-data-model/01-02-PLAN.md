    description: "实现自定义树模型，将 SNoteItem 树适配到 QTreeView。隐藏根节点模式。"
    behavior: |
      TreeModel 契约行为（必须通过测试验证）：
      - index(0, 0, QModelIndex()) 返回根的第一个子节点的有效 QModelIndex

      - flags(index) 返回 ItemIsEnabled | ItemIsSelectable
      - rowCount(column>0_parent) 返回 0（Qt 约定守卫）
      - headerData(0, Horizontal, DisplayRole) 返回 "笔记分区"
      
      数据变更行为：
      - add_item(parent_index, item) 正确插入子节点并发出信号
      - remove_item(index) 正确删除节点并发出信号
    implementation: |

         - index(): hasIndex 守卫, 解析 parent, createIndex 返回
         - parent(): 对根节点返回 QModelIndex(), 否则 find_parent + createIndex
         - rowCount(): column>0 守卫, 返回 parent 的 children 数

         - add_item(): beginInsertRows + append + endInsertRows
         - remove_item(): beginRemoveRows + del + endRemoveRows
