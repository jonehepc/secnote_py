# SecNotepad

## What This Is

一个本地加密笔记本应用，提供分区树 + 页面列表的层级笔记管理。笔记内容以加密文件形式存储在本地，用户通过密钥打开。富文本编辑、标签和搜索是核心编辑功能。面向需要安全存储笔记的个人用户使用。

## 核心价值

**笔记内容始终以加密状态保存在磁盘上，只有持有密钥的用户才能解密和阅读。**

## 上下文

**技术栈：** Python 3 + PySide6（Qt6 绑定），选择 PySide6 而非 PyQt6 是因 LGPL 授权对分发更友好。

**旧项目参考：** `/home/jone/projects/SafetyNotebook` — 原作者用 C++/Qt5 编写的安全笔记本 v0.1.2。核心思路延续到新项目：
- 数据模型：树形 SNoteItem（分区 + 笔记），JSON 序列化
- 加密方案：SHA-512(密码) 派生密钥 → AES 加密整体 JSON 文件
- 存储格式：单文件 `.secnote`
- 编辑器：QTextEdit 原生富文本，内容存 HTML

**新项目改进方向：**
- C++ → Python/PySide6，降低维护门槛
- 分区与页面分离展示（旧项目在同一个树视图中混合显示）
- 新增标签和搜索功能
- 现代化 UI 设计（Qt6 的样式表/QSS 美化）
- Qt5 → Qt6

**目标平台：** Windows 10+、Linux（Ubuntu/Debian 系及国产化系统如 UOS、Kylin）

## 约束

- **技术栈：** Python 3 + PySide6 + Qt6
- **加密库：** 使用 Python `cryptography` 库（而非自行实现 AES）
- **富文本：** QTextEdit 原生方案，不使用 QWebEngine（国产系统兼容性）
- **平台：** Windows + Linux 桌面，不包含 macOS
- **语言：** 代码和配置使用英文，面向用户的文档使用中文

## 关键决策

| 决策 | 理由 | 结果 |
|------|------|------|
| PySide6 而非 PyQt6 | LGPL 授权对分发无限制 | — 待验证 |
| QTextEdit 而非 QWebEngine | 国产 Linux 系统 WebEngine 兼容性不确定，原生组件更稳定 | — 待验证 |
| 整体文件加密而非逐页加密 | 实现简单，加入/退出时一次解密/加密 | — 待验证 |
| Python cryptography 库替代自定义 AES | 避免自行维护加密代码，安全性更好 | — 待验证 |
| 分区树 + 页面列表分离布局 | 相比旧项目的一棵树混合显示，结构更清晰 | — 待验证 |
