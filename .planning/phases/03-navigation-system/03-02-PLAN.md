  <behavior>
    核心行为 — 平铺展示分区下的 note 子节点列表：

    rowCount:
    - 场景 1: set_section 传入含 2 个 note + 1 个 section 子节点的分区 → rowCount() == 2（只计 note）
    - 场景 2: set_section 传入无 note 子节点的分区 → rowCount() == 0
    - 场景 3: set_section(None) → rowCount() == 0

    data (DisplayRole / EditRole):
    - 场景 4: data(index 0, Qt.DisplayRole) → 返回第一个 note 的 title 字符串
    - 场景 5: data(index 0, Qt.EditRole) → 返回第一个 note 的 title 字符串（与 DisplayRole 相同）
    - 场景 6: data(invalid_index, Qt.DisplayRole) → 返回 None
    - 场景 7: data(valid_index, Qt.DecorationRole) → 返回 None（未实现角色）

    flags:
    - 场景 8: flags(valid_index) → 包含 Qt.ItemIsEnabled | Qt.ItemIsSelectable | Qt.ItemIsEditable
    - 场景 9: flags(invalid_index) → 返回 Qt.NoItemFlags

    setData (EditRole 重命名, D-60):
    - 场景 10: setData(valid_index, "新标题", Qt.EditRole) → 返回 True，note.title 更新为 "新标题"，dataChanged 信号发射
    - 场景 11: setData(valid_index, "", Qt.EditRole) → 返回 False，note.title 不变 (D-60 空名称拒绝)
    - 场景 12: setData(valid_index, "   ", Qt.EditRole) → 返回 False，note.title 不变 (纯空格拒绝)
    - 场景 13: setData(invalid_index, "test", Qt.EditRole) → 返回 False
    - 场景 14: setData(valid_index, "test", Qt.DisplayRole) → 返回 False（非 EditRole）
    - 场景 15: setData(valid_index, 12345, Qt.EditRole) → 返回 False（非 str 类型）

    add_note:
    - 场景 16: add_note(SNoteItem.new_note("新页面")) → 返回 True，rowCount 从 N 变为 N+1，新 note 同时加入 self._section.children
    - 场景 17: set_section 为 None 时调用 add_note → 返回 False，rowCount 不变

    remove_note:
    - 场景 18: remove_note(valid_index) → 返回 True，rowCount 从 N 变为 N-1，note 从 self._section.children 中移除
    - 场景 19: remove_note(invalid_index) → 返回 False

    note_at:
    - 场景 20: note_at(valid_index) → 返回对应的 SNoteItem 对象
    - 场景 21: note_at(invalid_index) → 返回 None

    set_section 重置:
    - 场景 22: 两次连续 set_section（不同分区）→ 第二次替换第一次的 _notes 列表，rowCount 反映新分区的 note 数量

    关键约束 (Pitfall 5 — 对象引用一致性):
    - _notes 列表中的对象 == _section.children 中的同一对象（引用相同，非拷贝）
    - setData 修改 note.title 同时影响 _notes 缓存和 _section.children 源数据

  <implementation>
    实现 PageListModel 类：

    1. 继承 QAbstractListModel
    2. __init__:
       - super().__init__(parent)
       - self._section: SNoteItem | None = None
       - self._notes: list[SNoteItem] = []

    3. set_section(section: SNoteItem | None):
       - self.beginResetModel()
       - self._section = section
       - 如果 section 不是 None:
         self._notes = [c for c in section.children if c.item_type == "note"]
         注意：使用列表推导创建新列表，但列表中的元素是 section.children 的同一对象引用（不是深拷贝）
       - 否则:
         self._notes = []
       - self.endResetModel()

    4. rowCount(parent=QModelIndex()) → int:
       - 返回 len(self._notes)

    5. data(index, role=Qt.DisplayRole):
       - 检查 index.isValid() 和 index.row() < len(self._notes)
       - note = self._notes[index.row()]
       - 如果 role 是 Qt.DisplayRole 或 Qt.EditRole → 返回 note.title
       - 否则 → 返回 None

    6. flags(index) → Qt.ItemFlags:
       - 无效索引 → Qt.NoItemFlags
       - 有效索引 → Qt.ItemIsEnabled | Qt.ItemIsSelectable | Qt.ItemIsEditable

    7. setData(index, value, role=Qt.EditRole) → bool:
       - 验证 role == Qt.EditRole，index.isValid()，index.row() < len(self._notes)
       - D-60 验证: isinstance(value, str) 且 value.strip() != ""
         如果失败 → 返回 False（Qt 视图自动还原旧值）
       - note = self._notes[index.row()]
       - note.title = value.strip()  (Pitfall 5: 直接修改，因为 _notes[i] IS _section.children[j] 同一对象)
       - self.dataChanged.emit(index, index, [Qt.DisplayRole, Qt.EditRole])
       - 返回 True

    8. add_note(note: SNoteItem) → bool:
       - 如果 self._section is None → 返回 False
       - row = len(self._notes)
       - self.beginInsertRows(QModelIndex(), row, row)
       - self._section.children.append(note)  # 同步更新源数据
       - self._notes.append(note)
       - self.endInsertRows()
       - 返回 True

    9. remove_note(index: QModelIndex) → bool:
       - 验证 index.isValid() 和 index.row() < len(self._notes)
       - row = index.row()
       - note = self._notes[row]
       - 如果 note in self._section.children: self._section.children.remove(note)  # 从源数据中移除
       - self.beginRemoveRows(QModelIndex(), row, row)
       - self._notes.pop(row)  # 从缓存中移除
       - self.endRemoveRows()
       - 返回 True

    10. note_at(index: QModelIndex) → SNoteItem | None:
        - 验证 index.isValid() 和 index.row() < len(self._notes)
        - 返回 self._notes[index.row()] 或 None

    - 方法 docstring 说明用途和引用的决策编号

    - 不支持 QPersistentModelIndex — 新建后自动选中在 Plan 04 由 MainWindow 通过 QItemSelectionModel.select() 实现
