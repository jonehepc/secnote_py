<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SearchDialog → MainWindow navigation | 搜索结果对象请求改变主窗口选择状态 |
| Search action lifecycle → decrypted notebook | 弹窗 root 指向当前已解密内存树 |
| Navigation selection → editor display | 跳转将目标页面正文和标签加载到右侧编辑区 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-05-01 | T | `_select_note` | mitigate | 使用对象引用 `is` 找目标，正确经 `SectionFilterProxy.mapFromSource()` 映射，避免选错同名/同 ID 页面。 |
| T-05-05-02 | R/T | dirty-state | mitigate | 搜索入口、展示和跳转不调用 `mark_dirty()`；测试断言 `_is_dirty is False`。 |
| T-05-05-03 | I | dialog root/session | mitigate | `_clear_session()` 禁用搜索并清空/关闭 dialog root，避免旧笔记本明文对象被弹窗引用。 |
| T-05-05-04 | D | invalid stale result | mitigate | 目标不存在时只显示固定状态栏提示，不崩溃、不打印敏感数据。 |
</threat_model>
