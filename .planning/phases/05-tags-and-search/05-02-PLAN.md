| TagBarWidget → MainWindow | tag_added/tag_removed 信号将驱动页面元数据修改 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-02-01 | T | TagBarWidget input | mitigate | 对输入 trim；拒绝空值、超过 32 字符和当前页面内 casefold 重复；保留合法原始显示文本。 |
| T-05-02-02 | D | Tag chip layout | mitigate | 标签长度限制 32 字符；chip 可换行或至少不横向撑破主窗口。 |
| T-05-02-03 | I | TagBarWidget | mitigate | 组件不写文件、不记录日志、不接触加密保存；仅通过 Signal 传递当前用户输入。 |
| T-05-02-04 | T | HTML content boundary | mitigate | 组件不导入 `SNoteItem`、不修改 `SNoteItem.content`、不调用 `mark_dirty()`；标签数据边界留给 MainWindow。 |
</threat_model>
