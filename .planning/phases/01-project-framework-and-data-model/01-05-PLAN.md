# Phase 01 Plan 05: MainWindow UI Implementation

## Objective

实现 SecNotepad 的主窗口 UI 层，包含：欢迎页 (WelcomeWidget)、三栏布局（左侧分区树 QTreeView、中间页面列表 QListView、右侧编辑区占位）、菜单栏/工具栏/状态栏、以及 QStackedWidget 在欢迎页与三栏布局之间切换。

## Context

数据模型层 (SNoteItem、TreeModel、Serializer、Validator) 已在 Plan 01-02 中完成并通过验证（93/93 测试通过）。`__main__.py` 已在 Plan 01-01 中创建并 `from .ui.main_window import MainWindow`。本计划实现 UI 层使应用可运行。

依赖：
- `src/secnotepad/model/` — SNoteItem、TreeModel
- `src/secnotepad/app.py` — create_app()
- `src/secnotepad/__main__.py` — 入口（已引用 MainWindow）

### Task 3: 创建 MainWindow (auto)

Create `src/secnotepad/ui/main_window.py` with:

**窗口设置 (D-19):**
- 标题 "SecNotepad"，初始大小 1200x800，最小 800x600

**菜单栏 (D-16):**
- 文件菜单：新建(Ctrl+N)、打开(Ctrl+O)、保存(Ctrl+S, 灰显)、另存为(Ctrl+Shift+S, 灰显)、退出(Ctrl+Q)
- 编辑菜单：撤销/重做/剪切/复制/粘贴 (全部灰显)
- 视图菜单：切换面板显示 (灰显)
- 帮助菜单：关于 (灰显)

**工具栏 (D-17):**
- 新建、打开、保存(灰显)、另存为(灰显)
- 使用 QStyle.StandardPixmap 标准图标
- 不可移动

**中央区域 (D-14, D-15):**
- QStackedWidget: index 0 = 欢迎页, index 1 = 三栏 QSplitter
- 默认显示欢迎页 (index 0)

**三栏布局 (D-12, D-13):**
- QSplitter(Qt.Horizontal)
- 左侧 QTreeView (最小 100px, 可折叠)
- 中间 QListView (最小 100px, 始终可见)
- 右侧 QWidget 编辑区占位
- 初始宽度 [200, 250, 750]

**状态栏 (D-18):**
- 显示 "就绪"

**新建笔记本处理:**
- `_on_new_notebook()` 方法
- 创建 SNoteItem.new_section("根分区")
- 创建 TreeModel，绑定到 QTreeView
- 切换到三栏布局 (setCurrentIndex(1))
- 状态栏显示 "新建笔记本 - 未保存"

**连接:**
- 菜单新建、工具栏新建、欢迎页新建按钮 → _on_new_notebook
- 退出菜单 → self.close
- 打开按钮/菜单暂不绑定（Phase 2 实现）

**重要:**
- 所有 QAction 存储为实例变量防止 GC (陷阱 5)
- 参照 PATTERNS.md 的 MainWindow 模式和 RESEARCH.md 第 5 节代码

**Done criteria:**
- MainWindow 类存在且可创建
- 菜单栏有 4 个菜单（文件/编辑/视图/帮助）
- 工具栏有 4 个按钮
- QStackedWidget 有两个页面
- QSplitter 有三栏
- 新建笔记本按钮/菜单触发切换到三栏布局
- `__main__.py` 能通过 MainWindow 启动
