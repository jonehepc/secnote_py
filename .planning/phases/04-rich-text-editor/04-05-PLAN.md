<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MainWindow menu shortcuts → RichTextEditorWidget | 菜单/快捷键事件可能在无页面状态触发，必须按页面状态禁用或安全 no-op。 |
| RichTextEditorWidget content_changed → SNoteItem.content | 编辑器 HTML 写回当前页面，是核心数据完整性边界。 |
| Page switch → QTextDocument | 页面切换必须避免 dirty 误触发与 undo 栈跨页面泄漏。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-17 | Tampering | MainWindow edit actions | mitigate | 无页面时编辑 action 全禁用；有页面时只路由到 `_rich_text_editor`，避免快捷键误改导航数据。 |
| T-04-18 | Information Disclosure | `_show_note_in_editor` | mitigate | 使用 `load_html()` 清理 undo/redo stacks，测试跨页面撤销不能恢复上一页内容。 |
| T-04-19 | Integrity | `_on_editor_content_changed` | mitigate | 仅当前 editor stack 为编辑器页且当前 note 存在且 HTML 不同时写回并 mark_dirty。 |
| T-04-20 | Information Disclosure | zoom menu | mitigate | 缩放 handler 不访问 Serializer/SNoteItem，不写入 HTML，仅状态栏提示百分比。 |
</threat_model>
