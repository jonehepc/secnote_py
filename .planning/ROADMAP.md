## 阶段详情

### Phase 1：项目框架与数据模型

**目标：** 搭建 Python + PySide6 项目骨架，实现 SNoteItem 数据模型，创建基础主窗口。

**需求：** FILE-01, NAV-01（数据模型部分）

**计划：** 2 个计划（已完成）

**成功标准：**
1. 项目可通过 `python -m secnotepad` 启动并显示空白主窗口
2. SNoteItem 支持分区和笔记两种类型，可嵌套子节点，可序列化为 JSON 并反序列化还原
3. 主窗口展示三栏布局骨架（分区区、页面区、编辑区占位）

### Phase 2：文件操作与加密

**目标：** 实现 `.secnote` 文件的加密读写，完整的"新建→编辑→保存→打开→解密"流程。

**需求：** FILE-02, FILE-03, FILE-04, FILE-05, CRYPT-01, CRYPT-02, CRYPT-03, CRYPT-04

**计划：** 4 个计划 | 2 个 TDD + 2 个标准执行

**成功标准：**
1. 用户新建笔记本后可输入内容、保存为 `.secnote` 文件（需输入密码）
2. 打开已有 `.secnote` 文件时，输入正确密码可解密加载，错误密码显示错误提示
3. 文件内容使用 PBKDF2-SHA256(密码) + AES-256-CFB + HMAC-SHA256 加密存储，无法直接明文读取
4. 另存为功能可用，支持更改路径和密码
5. 关闭时若有未保存更改，弹出保存提示

**计划列表：**

- [x] 02-02-PLAN.md — 密码对话框 UI：PasswordDialog（3 种模式）+ PasswordGenerator
- [x] 02-03-PLAN.md — (TDD) 加密文件服务：PBKDF2+AES-256-CFB+HMAC-SHA256 的 encrypt/decrypt/save/open
- [x] 02-04-PLAN.md — MainWindow 集成：dirty flag、closeEvent、文件操作 handlers、最近文件列表

### Phase 3：导航系统

**目标：** 实现分区树形结构和页面列表，支持增删改操作。

**需求：** NAV-01, NAV-02, NAV-03, NAV-04

**成功标准：**
1. 分区以 QTreeView 展示，支持多级嵌套
2. 点击分区在中间面板显示该分区下的页面列表
3. 用户可创建/重命名/删除分区，删除含子内容的区时给出警告
4. 用户可在分区下创建/重命名/删除页面，点击页面在编辑区加载内容

**计划：** 4 个计划 | 2 个 TDD + 2 个标准执行

### Phase 4：富文本编辑器

**目标：** 实现完整的富文本编辑功能，包括格式工具栏和所有格式化操作。

**需求：** EDIT-01, EDIT-02, EDIT-03, EDIT-04, EDIT-05, EDIT-06, EDIT-07, EDIT-08, EDIT-09, EDIT-10, EDIT-11

**成功标准：**
1. 选中页面后编辑区可用，支持 QTextEdit 富文本编辑，内容以 HTML 存储
2. 格式工具栏包含：字体、字号、加粗、斜体、下划线、删除线、文字颜色、背景色
3. 支持段落对齐（左/中/右/两端）和标题（H1-H6）
4. 支持列表（有序/无序/待办复选框）和缩进调整
5. 撤销/重做和剪切/复制/粘贴功能正常

### Phase 5：标签与搜索

**目标：** 实现页面标签管理和全文搜索功能。

**需求：** TAG-01, TAG-02, TAG-03, SRCH-01, SRCH-02, SRCH-03

**成功标准：**
1. 用户可为页面添加多个标签，标签在页面上可见
2. 用户可移除页面的标签
3. 搜索栏输入关键词后，搜索页面标题和 HTML 内容，结果列表显示标题和高亮片段
4. 点击搜索结果可跳转到对应页面
5. 标签和搜索结果在数据模型中有持久化支持

### Phase 6：界面美化与收尾

**目标：** 应用 QSS 样式表实现现代化视觉风格，打磨交互细节。

**需求：** UI-01, UI-02, UI-03, UI-04, UI-05, UI-06

**成功标准：**
1. 三栏布局可调整大小，各面板比例合理
2. 工具栏和格式栏图标清晰、布局合理
3. 整体 QSS 主题现代化（配色统一、圆角、阴影等）
4. 状态栏实时显示操作反馈
5. 窗口标题正确显示文件名和修改状态标记
6. 关于对话框可正常显示
