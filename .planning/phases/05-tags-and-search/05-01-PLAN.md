<interfaces>
`SNoteItem` 已有字段：`id: str`、`title: str`、`item_type: str`、`content: str`、`children: list[SNoteItem]`、`tags: list[str]`。搜索服务不得修改这些字段。

创建以下公开契约：
```python
@dataclass(frozen=True)
class SearchFields:
    title: bool = True
    content: bool = True
    tags: bool = False

@dataclass(frozen=True)
class SearchResult:
    note: SNoteItem
    title: str
    section_path: str
    matched_field: str
    snippet: str

class SearchService:
    @staticmethod
    def html_to_plain_text(html: str) -> str: ...
    @staticmethod
    def search(root: SNoteItem | None, query: str, fields: SearchFields | None = None) -> list[SearchResult]: ...
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 解密内存数据 → 搜索服务 | 已解密的页面标题、HTML 正文和标签进入只读搜索逻辑 |
| 搜索服务 → UI rich text 片段 | 用户内容被包装成高亮片段，可能被 QLabel/结果项以 rich text 展示 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01-01 | I | SearchService | mitigate | 不创建外部索引、数据库、缓存文件或搜索历史；源码 grep 禁止 `open(`、`Path(`、`sqlite`、`QSettings`。 |
| T-05-01-02 | T/I | snippet generation | mitigate | 正文先通过 `QTextDocumentFragment` 转纯文本，用户内容先 `html.escape()`，只插入受控 `<mark>`。 |
| T-05-01-03 | D | tag/query length | accept | 搜索 query 不落盘且当前本地笔记本规模小；UI 层会限制标签长度，服务对空 query 返回空结果。 |
| T-05-01-04 | R/I | logging | mitigate | 服务不使用 `print` 或 `logging` 输出 query、片段、正文、标签。 |
</threat_model>
