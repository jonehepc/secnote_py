<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Qt signal/event → MainWindow handler | 用户事件和模型事件进入应用数据变更逻辑 |
| Model dataChanged → dirty state | 模型标题变更跨越模型层到窗口保存状态 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-10 | Denial of Service | `_setup_navigation()` repeated initialization | mitigate | `_teardown_navigation()` 清理 selection、按钮、菜单、action、shortcut 和 dataChanged 连接，防止重复触发造成指数级 handler 调用。 |
| T-03-11 | Tampering | Rename dirty-state path | mitigate | TreeModel/PageListModel `dataChanged` 接入 `_on_structure_data_changed()`，所有标题重命名都会调用 `mark_dirty()`，避免未保存结构变更丢失。 |
| T-03-12 | Tampering | New child section selection | mitigate | add_item 后使用 proxy parent 定位新增 child index 并 setCurrentIndex，避免后续新建页面/删除作用在错误父节点。 |
</threat_model>
